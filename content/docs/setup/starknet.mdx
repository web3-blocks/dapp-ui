To build StarkNet-powered features in your dApp, **dApp/ui** recommends using:

- [**Starknet React**](https://www.starknet-react.com/docs/getting-started) — React hooks for StarkNet apps.
- [**@starknet/web3.js / starknet.js**](https://www.starknet.io/blog/develop-dapp-starknet-js/) — TypeScript interface for StarkNet RPC & smart contracts.
- [**TanStack Query**](https://tanstack.com/query/v5) — An async state-manager that handles requests, caching, and more.
- [**TypeScript**](https://wagmi.sh/react/typescript) — Optional, but highly recommended.

dApp/ui components are fully compatible with these tools.

## Starknet React Setup

Starknet React. Reactivity for StarkNet apps.

---

### Install Dependencies

Before installing any StarkNet components, set up the required dependencies:

```bash
npm install @starknet-react/chains @starknet-react/core starknet @tanstack/react-query
```

---

### Create StarkNet Config

Create a client configuration file to set up the connection, chains, and wallet connectors.

```ts title="config/starknet.config.ts" showLineNumbers
import { mainnet, sepolia } from "@starknet-react/chains"
import { StarknetConfig, publicProvider } from "@starknet-react/core"

export const chains = [mainnet, sepolia]

export const provider = publicProvider({ chains })

// prepare wallet connectors (injected wallets like Braavos, ArgentX)
import { braavos, argentX } from "@starknet-react/core"

export const connectors = [braavos({ chains }), argentX({ chains })]

export const starknetConfigProps = {
  chains,
  provider,
  connectors,
  autoConnect: true,
}
```

In this example:

- StarkNet is configured for Mainnet and Sepolia via `@starknet-react/chains`.
- A public RPC provider is used.
- Wallet connectors include Braavos and ArgentX.

### Context Provider

Create a wrapper component that uses `StarknetConfig` to wrap your app.

```tsx {4} {5} {16} {17} {18} title="provider/starknet.provider.tsx" showLineNumbers
"use client"

import React from "react"
import { StarknetConfig } from "@starknet-react/core"
import { starknetConfigProps } from "@/config/starknet.config"
import { QueryClient, QueryClientProvider } from "@tanstack/react-query"

const queryClient = new QueryClient()

export default function StarknetProvider({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <StarknetConfig {...starknetConfigProps}>
      <QueryClientProvider client={queryClient}>{children}</QueryClientProvider>
    </StarknetConfig>
  )
}
```

---

### Wrap App in Context Provider

Wrap your app with the `StarknetProvider` component you created earlier.

```tsx {1} {11} title="app/layout.tsx" showLineNumbers
import StarknetProvider from "@/provider/starknet.provider"

export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode
}>) {
  return (
    <html lang="en" suppressHydrationWarning>
      <body>
        <StarknetProvider>{children}</StarknetProvider>
      </body>
    </html>
  )
}
```
