## Type Safety

All components and hooks are fully typed with TypeScript, ensuring type safety throughout your application.

### Connect Component Types

```ts showLineNumbers
import { useEthereum } from "@web3-blocks/dapp-ui"

// Returns strictly-typed slices
const {
  account, // AccountState
  connect, // ConnectState
  disconnect, // DisconnectState
  network, // NetworkState
} = useEthereum()

// Key fields
connect.isWalletAvailable // boolean — runtime EIP-1193 provider check
await connect.fn() // Promise<void> — initiates wallet connection
await disconnect.fn() // Promise<void> — disconnects current wallet
await network.switch(1) // switches to a supported chain by id
```

### Type Guards

```ts showLineNumbers
import {
  getCurrentChainInfo,
  getSupportedChainById,
  getUnsupportedChainMessage,
  getWalletAvailability,
  hasEthereumProvider,
  isChainSupported,
} from "@web3-blocks/dapp-ui"

// Minimal EIP-1193 provider shape check
if (!hasEthereumProvider(window.ethereum)) {
  // guide the user to install a wallet
}

// Runtime wallet availability
const isAvailable = getWalletAvailability()

// Chain helpers
const supported = isChainSupported(1, supportedChains)
const chain = getSupportedChainById(1, supportedChains)
const message = getUnsupportedChainMessage(1, supportedChains)
const info = getCurrentChainInfo(1, supportedChains) // { isSupported, chain?, message? }
```

## Overview

- `Button` is a styled primitive with accessible focus states and size/variant options.
- `Connect` and `Disconnect` are specialized buttons wired to `useEthereum()` for wallet actions.
- All slices from `useEthereum()` are fully typed for safe usage and auto-complete.

## Connect Button Behavior

- Disabled when: `loading`, `props.disabled`, or `!connect.isWalletAvailable`.
- Shows a spinner when `loading` or `isLoading`.
- Provides a helpful tooltip/title when no wallet is detected.

```tsx showLineNumbers
import { Wallet } from "lucide-react"

import { Connect } from "@/registry/new-york/ui/button"

export function ConnectExample() {
  return (
    <Connect
      variant="outline"
      loadingTxt="Connecting..."
      title="Connect Wallet"
    >
      <span>Connect Wallet</span>
      <Wallet />
    </Connect>
  )
}
```

## Best Practices

- Use `aria-label` on icon-only buttons for accessibility.
- Prefer `asChild` to make links or custom components look like buttons.
- Keep network actions responsive: show `isLoading` and concise text like `Connecting...`.
- Tailwind v4 sets button cursor to `default`. If you want `cursor: pointer`, add the base layer rule in your CSS.
